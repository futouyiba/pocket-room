## 技术亮点

### 1. 双重认证分离
- Gate Auth（用户身份）与 Provider Binding（AI 服务商）完全解耦
- 灵活的 OAuth 2.0 + PKCE 实现
- 自动 Token 管理和刷新

### 2. 资源可控的 AI 协作
- 四阶段治理生命周期（Summon → Request → Approve → Respond）
- 显式上下文选择，防止过度消耗 token
- 审批豁免机制（Owner、白名单）

### 3. 隐私优先的消息可见性
- 后加入成员仅能看到加入后的消息
- RLS 策略强制执行
- 通过 Segment 分享历史上下文

### 4. 灵活的 Room 加入策略
- 三种互斥模式（申请审批、自由加入、密码加入）
- 丰富的审批操作（批准、拒绝、封禁、静默）
- 被邀请人加入特权

### 5. 全面的测试覆盖
- 46 个属性测试（每个 100 次迭代）
- 完整的单元测试和集成测试
- E2E 测试覆盖关键用户流程
- 测试覆盖率 > 85%

### 6. 优秀的用户体验
- 实时消息推送
- 降级策略和自动重试
- 用户友好的错误提示
- 响应式设计
