# Sprint2：Practice Room（练功房）

## 一个场景

- 一个新人
- 一个有经验的朋友
- 一个共享的上下文
- 一个正在 build 的东西

他们不一定要产出商业级产品，
但他们在一起真正理解了一个问题。

## Practice Room 是什么？

一个长期存在的结对空间

- 可以实时语音（对讲或持续）
- 可以一起看代码、看上下文
- 可以把讨论沉淀为 Pocket 里的 Segment

## 师徒关系不是权限关系

没有"控制"

没有"审批"

只有"共享上下文"

上下文本身，就是最好的教学。

## 为什么语音重要？

因为有些理解：

- 写不出来
- 打字太慢
- 但说出来刚刚好

## Practice Room 允许：

- 按住说话
- 同时被对方听见
- 同时被记录、整理
